Setup Continuous Integration with Azure DevOps

Azure DevOps is a CI/CD solution from Microsoft which seemlessly integrates with Azure. In his blog post Azure DevOps Continuous Build/Deploy/Test with ASP.NET Core 2.2 Preview in One Hour, Scott Hanselman guides us through his process of setting up Azure DevOps for his podcast website.

And another important point: Azure DevOps is free for Small Teams and Open Source projects. OS projects even geht unlimited minutes for their CI/CD. So definitely give it a try!
